## ENV
Crear archivo .env e ingresar las variables de .env-template

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Upload the info

You must have pentaho and use etl transactional ETL documents into pentaho to extract the info
